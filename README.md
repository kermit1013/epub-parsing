# Epub TTS (EPUB 轉語音工具)

這是一個將 EPUB 電子書自動轉換為語音檔案（WAV）的 Node.js 終端機工具。它會讀取並解析 EPUB 檔案內容、過濾多餘字元、進行文本分段，並透過 TTS API 將文字轉換成音檔。

**🚧 備註：目前本工具的語音合成後端支援 Qwen3 模型。**

## 功能特色

- **解析 EPUB**：讀取 EPUB 格式電子書並提取純文字。
- **文本前處理**：自動刪除雜訊字元，並使用 `OpenCC` 進行繁體轉簡體轉換，確保模型朗讀的準確度。
- **智慧分段**：依據句子標點符號與字數長度限制（預設 200 字）將文本切塊，避免單一句子過長導致 TTS 逾時，但確保語意完整。
- **TTS 語音生成**：呼叫本地端的 TTS API（目前支援僅 **Qwen3**），自動依序將文章段落轉為 `.wav` 音檔。
- **中斷點接續轉換**：若中途轉換失敗或手動停止，下次可直接指定「起始章節」繼續轉換，無需重頭開始。
- **一鍵 iCloud 同步**：可將轉好的所有音檔自動複製到你的 iCloud 資料夾（預設支援 iOS 的 BookPlayer App 路徑），方便手機直接閱聽。

## 系統需求與安裝

1. 確保你的系統已安裝 **Node.js** (建議 v18 以上版本)。
2. 複製專案並安裝依賴套件：

```bash
npm install
```

3. 由於需要呼叫 TTS 服務（支援 Qwen3），請確保你的本地端已啟動相關的 TTS API 伺服器（預設監聽在 `http://127.0.0.1:8001/...`）。

## 使用方式

在終端機輸入以下指令啟動互動式選單：

```bash
node index.js
```

啟動後，使用上下方向鍵選擇要執行的動作：

### 1. 轉換 EPUB 為語音檔案 (Convert)
- 輸入 epub 檔案路徑（如：`input/book.epub`）。
- 指定從第幾章節開始轉換（如果在中途停止，可利用此功能接續轉換）。
- 輸入 TTS API URL（如果你的 Qwen3 API 位址不同，可於此處覆寫修改）。

轉換完成的音檔會保存在專案目錄底下的 `./output/<書名>/` 資料夾內。

### 2. 將已轉換的音檔同步至 iCloud (Sync)
- 輸入 epub 的檔案路徑或是書名。
- 輸入要同步的目標資料夾路徑。如果是 macOS 用戶，預設會提供 BookPlayer 的 iCloud 資料夾路徑：
  `~/Library/Mobile Documents/iCloud~com~tortugapower~audiobookplayer/Documents`

執行後會將 `output` 對應書名字資料夾內的 `.wav` 檔全數複製過去。

### 3. 離開 (Exit)
結束程式。
