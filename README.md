# Epub TTS (EPUB 轉語音工具) - CosyVoice 版

這是一個將 EPUB 電子書自動轉換為語音檔案（WAV）的 Node.js 終端機工具。它會讀取並解析 EPUB 檔案內容、過濾多餘字元、進行文本分段，並透過 **CosyVoice (Gradio API)** 將文字轉換成高品質音檔。

**🚧 備註：目前本工具已升級支援 CosyVoice 的「3s 極速複刻」模式，可完美還原參考音頻的音色。**

## 功能特色

- **解析 EPUB**：讀取 EPUB 格式電子書並提取純文字。
- **文本前處理**：自動刪除雜訊字元，並使用 `OpenCC` 進行繁體轉簡體轉換，確保 CosyVoice 模型朗讀的準確度。
- **智慧分段**：依據句子標點符號與字數長度限制（預設 200 字）將文本切塊，確保語音流暢且語意完整。
- **CosyVoice 整合**：
    - 支援 **3s 極速複刻**：只需提供 3~30 秒的參考音頻即可複刻音色。
    - 支援 **Gradio API**：透過 `@gradio/client` 直接連線到 CosyVoice 服務端。
    - **自動處理串流**：內建 m3u8 串流解析與片段合併邏輯，輸出的音檔為標準 WAV 格式。
- **中斷點接續轉換**：若中途轉換失敗或手動停止，下次可直接指定「起始章節」繼續轉換。
- **一鍵 iCloud 同步**：自動將轉好的音檔同步到 iCloud 資料夾（預設支援 iOS 的 BookPlayer App），方便隨時收聽。

## 系統需求與安裝

1. 確保你的系統已安裝 **Node.js** (建議 v18 以上版本)。
2. 確保你的 CosyVoice 服務已啟動並開啟 Gradio 介面。
3. 複製專案並安裝依賴套件：

```bash
npm install
```

## 使用方式

執行以下指令啟動：

```bash
node index.js
```

### 1. 轉換 EPUB 為語音檔案 (Convert)

啟動後需輸入以下資訊：
- **epub 檔案路徑**：例如 `input/mybook.epub`。
- **起始章節**：預設為 1。
- **CosyVoice Gradio URL**：預設為 `http://100.84.200.122:50000` (或你自己部署的位址)。
- **Prompt 音頻路徑**：作為音色參考的 WAV 檔案（建議 ≥16kHz）。
- **Prompt 文字**：該參考音頻對應的內容文字（建議使用簡體以獲得最佳效果）。
- **隨機種子 (Seed)**：
    - 輸入 `0`：每次轉換章節都會自動產生新的隨機種子。
    - 輸入特定數字：固定使用該種子以維持穩定的咬字風格。
- **語速 (Speed)**：預設為 `0.9`，提供最自然的聽感。

轉換後的音檔存放在 `./output/<書名>/` 資料夾。

### 2. 將音檔同步至 iCloud (Sync)
- 輸入書名或路徑。
- 同步至指定目標（預設為 macOS 的 iCloud BookPlayer 路徑）。

## 測試腳本

若想單獨測試 CosyVoice 的效果而不轉換整本書，可使用：
```bash
node test-cosyvoice.mjs --text "你要測試的文字"
```
